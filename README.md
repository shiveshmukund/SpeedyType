## 📌 About The Project

**SpeedyType** is a modern and responsive typing speed test application designed to help users improve their typing speed, accuracy, and consistency through real-time feedback and analytics.

The project is built entirely using **Vanilla JavaScript**, **HTML5**, and **CSS3** without external frameworks, focusing on performance, clean UI/UX, and efficient frontend architecture.

This project demonstrates:

* Real-time DOM manipulation
* State management
* Performance optimization
* LocalStorage integration
* Responsive UI development
* Event-driven programming
* Dynamic analytics rendering

## 📊 Core Metrics

### Words Per Minute (WPM)

WPM = \frac{\text{Characters Typed}/5}{\text{Time in Minutes}}

### Accuracy Formula

Accuracy = \frac{\text{Correct Characters}}{\text{Total Characters}} \times 100
