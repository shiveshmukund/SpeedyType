## 📱 User Experience Flow

1. **Initial Visit**
   - App loads with dark mode
   - Typing guide modal appears
   - User can read hand position guide

2. **Test Setup**
   - Select typing mode (words/sentences/paragraph)
   - Choose difficulty (beginner/intermediate/advanced)
   - Pick timer duration (15/30/60/120 seconds)
   - Select language (English/Hindi)
   - Close guide when ready

3. **Active Test**
   - Start typing when ready (timer auto-starts)
   - See real-time character highlighting
   - Watch WPM/accuracy update live
   - View progress bar filling
   - Receive coaching feedback

4. **Test Completion**
   - Timer auto-completes at 0
   - Results panel appears with stats
   - See error analysis and weak keys
   - Review performance metrics
   - Restart immediately or adjust settings

5. **Data Analysis**
   - Open analytics dashboard (Ctrl+I)
   - View speed trends chart
   - Check accuracy trends
   - Identify weak keys
   - Monitor career statistics
